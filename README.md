# License Plate Recognition
### Setup
  1. python environment: 'python -m venv env'
  2. install package: 'python -m pip install -r requirements.txt'

### Environment
  - enable: '.\env\Scripts\activate'
  - disable: 'deactivate'

### Run
  - 'python main.py'
